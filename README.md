 - 基于SpringCloud 2022、Spring Boot 2.4、OAuth2 的RBAC 权限管理系统

 ## 核心依赖
 | 依赖                   | 版本          |
|----------------------|-------------| 
| Spring Cloud         | 2021.0.1    | 
| Spring Boot          | 2.6.3       |
| Spring Cloud Alibaba | 2021.0.1.0  |
